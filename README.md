#   Walmart Sales Analysis Dashboard

##  Project Overview

This project analyzes Walmart sales data and builds a forecasting model to predict future sales trends. The goal is to derive business insights and support decision-making using data visualization and predictive analytics.

---

##  Tools Used

* Excel → Data Cleaning & Preprocessing
* Power BI → Interactive Dashboard

---

---

## Key Insights

- Sales show a declining trend over time  
- Los Angeles is the top-performing city  
- Profit margin is approximately 30%  
- Payment methods are evenly distributed  
- Product-level performance varies significantly  

---

## Data Analysis Approach

* Cleaned and transformed raw sales data using Excel
* Performed exploratory data analysis to identify trends
* Built an interactive Power BI dashboard
* Visualized key business metrics such as sales, profit, and city performance
 

---

##  Project Structure

* `data/` → cleaned dataset
* `dashboard/` → Power BI dashboard
* `images/` → dashboard preview

---

##  How to Use

1. Open `.pbix` file in Power BI Desktop
2. Explore interactive visuals
3. Adjust filters for deeper insights

---


## Business Impact

- Helps identify top-performing cities and products  
- Supports data-driven decision-making  
- Enables better understanding of sales and profit trends
